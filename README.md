# FalsisDev Api
Useful API made with JavaScript.

## Example Usage
```js
const fetch = require("node-fetch")
fetch(`https://api.falsisdev.ga/wiki?wiki=Api`).then(falsis => falsis.json()).then(api => {
api.summary.extract
})
```
